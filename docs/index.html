Aplicación Web Java para gestionar mensajes - Sección 17 y 18 del curso Udemy

🧠 Descripción

Este proyecto es parte del curso de Java con Maven y Servlets. La app permite crear, leer, actualizar y eliminar mensajes desde una interfaz web. En la Sección 18 se implementa conexión con MySQL.

🛠 Tecnologías usadas

  • Java 17
  • Maven
  • Servlets
  • MySQL
  • Bootstrap 5
  • GitHub
