> ### DESCRIPCIÓN  
> **Java: Curso profesional de Java – De cero a Master**  
> 
> Aprende a programar en Java desde cero hasta avanzado, incluyendo **Java SE**, **Java EE** y **Spring Boot**, utilizando **JDK 8**, **JDK 11** y **JDK 17**.
> 
> ---
> 
> 🔹 **Fundamentos de Programación en Java**  
> Java es un lenguaje de programación multiplataforma gracias a la JVM (Máquina Virtual de Java). Aprende los fundamentos y características esenciales del lenguaje para desarrollar tus propias aplicaciones:
> 
> - Conoce el lenguaje de programación Java  
> - Crea tu primer programa en Java  
> - Manejo de variables y operadores  
> - Estructuras de control  
> - Introducción a la programación funcional en Java  
> - Crea aplicaciones divertidas como juegos  
> 
> ---
> 
> 🔹 **Programación Orientada a Objetos (POO)**  
> Java fue creado para la Programación Orientada a Objetos, lo cual permite desarrollar sistemas más organizados y estructurados. Aprende los **cuatro pilares de la POO**:
> 
> - Encapsulamiento  
> - Abstracción  
> - Herencia  
> - Polimorfismo  
> 
> ---
> 
> 🔹 **Persistencia de Datos**  
> Aprende cómo guardar y mantener datos utilizando archivos y bases de datos como **MySQL** y **PostgreSQL**:
> 
> - Uso de colecciones y estructuras de datos  
> - Manejo de archivos en Java (crear, leer, escribir, eliminar)  
> - Conexión a bases de datos con JDBC  
> - Realizar operaciones CRUD (Create, Read, Update, Delete)
